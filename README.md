# Análisis Estadístico de Valores Numéricos

Este script de Python realiza un análisis estadístico básico en una lista de valores numéricos. Calcula varias métricas estadísticas que incluyen frecuencia absoluta, frecuencia acumulada, frecuencia relativa, frecuencia relativa acumulada, probabilidad y probabilidad acumulada en porcentaje.
## Funcionalidades

- **Frecuencia Absoluta (`fi`)**: Cuenta la ocurrencia de cada valor en la lista.
  
- **Frecuencia Acumulada (`Fi`)**: Calcula la suma acumulada de las frecuencias absolutas.

- **Frecuencia Relativa (`ri`)**: Calcula la proporción de cada valor en relación al tamaño total de la muestra.

- **Frecuencia Relativa Acumulada (`Ri`)**: Calcula la suma acumulada de las frecuencias relativas.

- **Probabilidad (`pi%`)**: Calcula la probabilidad de cada valor en porcentaje.

- **Probabilidad Acumulada (`Pi%`)**: Calcula la suma acumulada de las probabilidades en porcentaje.
